# 导航重构设计

## 概述

简化导航结构，明确双维度筛选逻辑，优化用户体验。

## 设计决策

| 决策点 | 选择 |
|--------|------|
| 导航模式 | 双维度筛选（内容类型 + 主题分类） |
| 筛选逻辑 | 交集过滤，空结果显示提示 |
| 分类命名 | 中文映射 |
| 筛选状态 | 融入标题区 |
| 侧边栏内容 | 功能入口 + 热门分类 |
| 功能入口 | 首页、收藏、RSS 源（3 项） |

## 侧边栏设计

### 结构

```
┌──────────────────┐
│ 导航             │  ← 小标题，弱化显示
│ ● 首页           │  ← 选中态：蓝色圆点 + 文字高亮
│ ○ 收藏           │
│ ○ RSS 源         │
├──────────────────┤
│ 分类             │
│ ○ 人工智能  (9)  │  ← 中文名 + 计数
│ ○ 编程开发  (3)  │
│ ○ 产品商业  (1)  │
│ ○ 其他      (4)  │
└──────────────────┘
```

### 分类映射表

```typescript
const CATEGORY_LABELS: Record<string, string> = {
  'ai-llm': '人工智能',
  'programming': '编程开发',
  'product-business': '产品商业',
  'other': '其他'
}
```

### 交互规则

- 点击「首页」→ 清除所有筛选，显示全部内容
- 点击分类 → 与顶部 Tab 形成交集筛选
- 点击「收藏」→ 显示收藏列表（MVP 可先占位）
- 点击「RSS 源」→ 切换到 RSS 源管理页

## 标题区筛选状态

### 状态示例

| 状态 | 显示 |
|------|------|
| 无筛选 | `精选推荐                    14 篇` |
| 仅内容类型 | `视频                        5 篇` |
| 仅主题分类 | `人工智能                    9 篇` |
| 双重筛选 | `人工智能 · 视频      3 篇 ｜ 清除筛选` |
| 空结果 | `人工智能 · 播客      0 篇 ｜ 清除筛选` + 空状态提示 |

### 实现要点

- 标题动态组合：`{分类} · {类型}` 或单独显示
- 「清除筛选」仅在有筛选时显示
- 点击清除 → 重置分类为空，Tab 回到「全部」
